## AmbiSecure One Pass — A Multiple Application Card

AmbiSecure One Pass is a "pocket-sized card that has an embedded microprocessor." The card "can do cryptographic operations, store, and process the digital credentials of the users securely."

### Use Cases
- Smart Door Access
- Computer Logical Access
- Enterprise Systems Login
- Share Contact Details With a Single Tap
- Payments

> "AmbiSecure One Pass for Enterprise Single Sign-On provides certificate-based strong authentication when users access their Credential Wallet by using smart cards."

The card functions as "a physical card that has an embedded integrated chip that acts as a security token." It matches "the same size as a driver's license or credit card and can be made out of metal or plastic."

## Key Benefits

### Stronger Hardware Backed Security
"Storing your credentials on a hardware key is safer than storing them on a mobile phone."

### Portable Credentials Across Devices
"Your credentials work seamlessly across multiple devices."

### Cross-platform Coverage
"AmbiSecure OnePass works across Windows, macOS, iOS and Android. Get the same set of codes across all platform for desktops as well as for all leading mobile platforms."

## Features

### Strong Authentication
"The AmbiSecure One Pass Card offers superior security by combining hardware-based authentication and public key cryptography to effectively defend against phishing attacks and eliminate account takeovers." Supports FIDO U2F and FIDO2.

### Easy, Fast, and Reliable
"The AmbiSecure One Pass Card provides a simple and intuitive authentication experience that users find easy to use, ensuring rapid adoption and organizational security with the click of a button." The card "does not require a battery making sure you never run out of juice to protect your IT infrastructure."

### Passwordless Enabled
"The AmbiSecure One Pass Card helps organizations accelerate to a passwordless future with resident credentials by providing support for the FIDO2 protocol. The protocol supports not only today's two-factor authentication but also paves the way for eliminating weak password authentication, with strong single or multi factor hardware-based authentication."

## Applets / Capabilities

### FIDO
"The AmbiSecure FIDO product is a hardware authentication solution that is FIDO Alliance certified. It helps organizations accelerate to a password-less future by providing support for the FIDO2 protocol."

### PIV
"The AmbiSecure ONE Pass Card is compatible with the PIV card interface, and are also fully interoperable with the Windows and Mac OS thus avoiding the need for a complex and expensive PIV management system."

### NDEF
"AmbiSecure One Pass Card lets you exchange contact info just by tapping. You can tap and share you contact details."

### Door Access
"The AmbiSecure ONE Pass Card is also a security token that grants you access through electrically-powered doors. You can gain access by just tapping your card on the reader."

### Certificate Storage
"AmbiSecure Certificate Storage provides facility to store any PEM or PFX certificate within a secure element for various use cases where secure storage of certificates is required."

### Secure Storage
"AmbiSecure Secure Storage Applet provides the user the ease to storage any kind of text for over 200 records inside the secure element that can be retrieved only after authorization of the Biometrics and/or PIN of the owner of the data."

### Open PGP
"The AmbiSecure ONE Pass Card can serve key-based encryption method used to encrypt files so that only their intended recipient can receive and decrypt them."
