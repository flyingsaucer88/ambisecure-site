## AmbiSecure Security key manager

> "AmbiSecure security key manager will enable use to personalise their key. injecting their own cert, pem files into the card. User can reset their fido credentials."

> "The AmbiSecure Security Key Manager helps the AmbiSecure ONE Pass USB Key, AmbiSecure ONE Pass Card as well as other FIDO devices to reset credentials. The tool also helps load the key pairs as well as the certificate compliant to the PIV standard into the AmbiSecure devices."
