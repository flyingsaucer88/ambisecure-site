## AmbiSecure Key — Security Digital USB Key

The AmbiSecure One Pass USB Key is a hardware-based authentication solution that meets FIDO2 standards. Designed to enable passwordless sign-in on FIDO-compatible servers, the product emphasizes hardware security as password reliance declines.

## Features

- **Easy to Use** — No battery or network connectivity required, users simply insert and tap to authenticate
- **Touch-to-Sign** — Establish user presence, and prevent remote takeovers, with a simple touch on the AmbiSecure Key during login
- **Security** — AmbiSecure Key provides the user a secure authentication
- **Efficient** — Reduce password reset or account lockout queries
- **Compatible** — AmbiSecure Key fits into any standard USB port
- **Portable** — Fit AmbiSecure Key on a keyring and carry without any worry

## Applets

### FIDO
The AmbiSecure FIDO product is a hardware authentication solution that is FIDO Alliance certified. It helps organizations accelerate to a password-less future by providing support for the FIDO2 protocol.

### PIV
The AmbiSecure ONE Pass Card is compatible with the PIV card interface and works with Windows and Mac OS, eliminating complex PIV management needs.

### NDEF
AmbiSecure One Pass Card lets you exchange contact info just by tapping. You can tap and share your contact details.

### Door Access
The AmbiSecure ONE Pass Card is also a security token that grants you access through electrically-powered doors. Tap to enter.

### Certificate Storage
AmbiSecure Certificate Storage provides facility to store any PEM or PFX certificate within a secure element for various use cases where secure storage of certificates is required.

### Secure Storage
AmbiSecure Secure Storage Applet provides the user the ease to storage any kind of text for over 200 records inside the secure element that can be retrieved only after authorization of the Biometrics and/or PIN of the owner of the data.

### Open PGP
The AmbiSecure ONE Pass Card can serve key-based encryption method used to encrypt files so that only their intended recipient can receive and decrypt them.
