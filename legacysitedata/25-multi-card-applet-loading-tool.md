## AmbiSecure Multi Card Applet Loading Tool

> "AmbiSecure multi card applet tool can load new applets on the cards and can delete, reset the card aswell. Using SCP02 authentication protocol."

The tool addresses the challenge that "loading of Applets with authentication like SCP02 or SCP03 is a tedious task." It enables:
- Updating default Card Manager keys
- Loading uniquely identifiable serial numbers
- Managing applets on AmbiSecure devices
