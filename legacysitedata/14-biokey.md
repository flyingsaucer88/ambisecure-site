## How Biometric Security Keys Work

### Step 1 — Biometric Enrollment
Register up to 100 fingerprints directly on the hardware security key. The fingerprint enrollment process takes seconds and stores encrypted biometric templates in the device's secure element. Ideal for shared device authentication, hot desking environments, and multi-user workstations.

### Step 2 — Hardware Authentication
Insert the AmbiSecure BioKey USB security device into any compatible port. The biometric sensor verifies your fingerprint against stored templates using on-device matching — no internet connection required, no cloud processing, complete offline biometric authentication.

### Step 3 — Instant Secure Access
Once verified, the cryptographic security key grants access to protected systems, applications, and data. Hardware-verified authentication provides phishing-resistant security that cannot be bypassed through social engineering, malware, or credential theft.
