## AmbiSecure – JavaCard Applet

> "AmbiSecure Enterprise Solution consists of multiple JavaCard Applet and various form factors of authenticators that are built with various use cases in mind and for different customer segments."

## Product Solutions

### FIDO
"The AmbiSecure FIDO product is a hardware authentication solution that is FIDO Alliance certified. It helps organizations accelerate to a password-less future by providing support for the FIDO2 protocol."

### PIV
"The AmbiSecure PIV solution is a NIST Compliant solution which is fully interoperable with the Windows, Linux and macOS thus avoiding the need for a complex and expensive PIV management system."

### Door Access
"AmbiSecure Door Access Solution works in conjunction with physical access solutions to grants you access through electrically-powered doors. You can gain access just by tapping your card on the reader."

### Certificate Storage
"AmbiSecure Certificate Storage provides facility to store any PEM or PFX certificate within a secure element for various use cases where secure storage of certificates is required."

### Open PGP
"AmbiSecure OpenPGP product servers as a key-based encryption method used to encrypt files so that only their intended recipient can receive and decrypt them."

### Business Card (NDEF)
"AmbiSecure NDEF Product lets you exchange contact info just by tapping on the other person's mobile device."

### OpenID Connect
"AmbiSecure OpenID Connect allows clients to verify the identity of the End-User based on the authentication performed by an Authorization Server, as well as to obtain basic profile information about the End-User in an interoperable and REST-like manner."

### SCP02
"The AmbiSecure SCP02 product implements the SCP02 authentication mechanism on a secure element to authenticate any smart card supporting the SCP02 authentication mechanism."

### FIDO Biometric
"The AmbiSecure FIDO Biometric product works on the AmbiSecure BioCard to use the person's fingerprint for a password-less future by providing support for the FIDO2 protocol."

### PIV Biometric
"The AmbiSecure PIV Biometric solution is a NIST Compliant solution which is fully interoperable with the Windows, Linux and macOS thus avoiding the need for a complex and expensive PIV management system."

### Secure Storage
"AmbiSecure Secure Storage Applet provides the user the ease to storage any kind of text for over 200 records inside the secure element that can be retrieved only after authorization of the Biometrics of the owner of the data."

### IoT Security (Key Manager)
"AmbiSecure Key Manager solution provides features like Set/Get Master Key, Generate Key Pair, Sign data etc. and some other proprietary implementations for specific use cases."
