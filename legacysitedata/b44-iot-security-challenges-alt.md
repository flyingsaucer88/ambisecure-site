## Introduction

> "IoT Security is one of the least talked about areas today since the world focuses more and more on Information security, Blockchain, and others."

> "Challenges with IoT security are many hence we have decided to break it down into two articles, this being the first of the two parts."

## 1. IoT Devices with Outdated Secure Firmware
- Security professionals previously focused on mobile devices and computers
- Over a billion IoT devices now communicate with enterprises
- Manufacturers prioritize building new devices over security updates
- Many devices never receive updates or receive insufficient ones
- Devices become vulnerable when hackers discover bugs, especially in open-source software
- Legacy systems interconnected with IoT devices lack modern security standards

## 2. Use of Weak and Default Credentials
- Many IoT companies provide default credentials (e.g., admin/password)
- Hackers use brute-force attacks with known usernames
- The Mirai botnet attack exploited default credentials
- Consumers are rarely informed to change default credentials

## 3. Lack of Encryption
> "Encryption is a great way to prevent hackers from accessing data."
- IoT devices lack storage and processing capabilities of traditional computers
- Unused bandwidth or processing power can be exploited for attacks
- Hackers can manipulate protective algorithms

## 4. Malware and Ransomware
- Cybercriminals have advanced techniques
- IoT cameras can be infected with malware, then encrypted
- Attackers demand payment to restore access
- City infrastructure malware can launch DDoS or MITM attacks

## 5. Predicting and Preventing Phishing Attacks
> "Phishing is already a security concern across all enterprise technologies. IoT devices represent the latest attack vector."
- Organizations fail to properly train workers on phishing threats
- Modern cloud services use threat intelligence for prediction
- AI-powered monitoring and analytics tools exist but are complex to adapt for IoT

## 6. Wide Area Networks (WAN)
- City-wide networks control communication for essential services
- Smart Meters and Smart Street Lights use WAN
- Uploading malware into one trusted device compromises entire enterprise security

## 7. Smart Home Devices
- More homes and offices gaining IoT connectivity
- Builders powering apartments with IoT devices
- Exposed IP addresses can reveal residential addresses and contact details
