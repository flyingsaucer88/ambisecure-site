Searchable reference databases for the bytes you stare at every day.

V2X / ITS PKI

IEEE 1609.2

IEEE 1609.2 V2X Certificates

Certificate structure, COER encoding, IssuerIdentifier, ToBeSignedCertificate fields, signature schemes, and the comparison table with X.509. Pair with the V2X parser tool to walk live bytes.

Structured reference → ETSI TS 103 097

ETSI TS 103 097 Secured Messages

SecuredMessage structure, header info fields, signer-info modes (digest, certificate, chain), application profile constraints, and the equivalence with IEEE 1609.2.

Structured reference → ETSI TS 102 941

ETSI TS 102 941 Trust & Privacy

V2X PKI hierarchy — Root CA, EA, AA, EC and PC issuance, Butterfly Key Expansion, CRL and CTL distribution. The architectural layer above 1609.2.

Structured reference → ISO 21177

ISO 21177 ITS Station Security

ITS-S session-layer security services, secure-session establishment, mutual authentication using 1609.2 certificates. The bridge from V2X PKI into application-layer secure sessions.

Structured reference → GSMA SGP.32

GSMA SGP.32 IoT eSIM

eUICC architecture, IPA, SM-DP+, SM-DS, IoT profile-discovery flow, and the parallel to V2X EA / AA lifecycle. The fleet-credential pattern at telecom scale.
